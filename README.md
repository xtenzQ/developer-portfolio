## üíª How to build the project?
Install dependencies
```Bash
npm install
```

Build sources
```Bash
npm run build
```

Develop with
```Bash
npm run serve
```

Website is built into `dist` directory.

‚ùó**Don't forget to change/remove my `CNAME` file before deployment!**
